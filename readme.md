# Football⚽️News

This project is a simple HTML and CSS practice project for creating a football news website. It is **not responsive** and is only intended for **large PC screens**.

## Project Description

This project includes a static HTML file (`index.html`) that represents the structure of the football news website. The styling is done using CSS, with the main styles located in the `styles/index.css` file.

## Features

- Navbar with Hamburger menu, logo, and search bar.
- Top headline news section with images and article content.
- Main content section including videos and recent posts.
- Aside section with Serie A and Champions League tables, as well as upcoming games.
- Footer with links to Terms of Service, Privacy Policy, and Contact Us.

## Usage

To view the project, simply open the `index.html` file in a web browser. Since it is not responsive, it is recommended to view it on a large PC screen.
